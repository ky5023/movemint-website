TransitEase は、人に寄り添う駅ナビゲーションを実現します。

ビジョン

わたしたちは「移動のストレスをなくす」ことを目指しています。 駅の複雑な構造、混雑、わかりにくい出口案内。 それらをオープンデータで見える化し、安心して移動できる世界をつくります。

課題

  • 駅構内の案内情報が統一されていない
  • Googleマップなどは「駅の中」を正確に案内できない
  • 高齢者や外国人、障がい者への配慮が不足
  • 実際の乗り換え時間とアプリ上の時間が合わない

解決策（TransitEaseの仕組み）

01

駅構内ナビゲーション

オープンデータから改札・階段・エレベーター・出口の位置をマッピング。 駅の「中」まで案内します。

02

リアルタイム混雑回避

混雑情報API（例：東京メトロ）を活用し、混雑が少ないルートを優先提案。 ストレスのない移動をサポートします。

03

目的別ルート提案

「トイレに寄りたい」「エレベーターを使いたい」「人が少ない道を通りたい」など、 利用者の状況に応じた案内を提供します。

オープンデータの活用

使用データ

  • GTFS / GTFS-RT（公共交通データ）
  • 駅施設データ（出口・改札・エレベーターなど）
  • 混雑情報API（東京メトロ・JR東日本）
  • OpenStreetMap / 国土地理院地図

目的

  • 駅構内の経路情報を統合
  • 混雑を可視化し、分散を促す
  • 観光や地域回遊の支援

価値提案

鉄道事業者向け

  • 混雑分散と利用者満足度の向上
  • リアルタイム人流データによる運行最適化

自治体向け

  • 観光案内や移動支援のデータ基盤に活用
  • 都市回遊データの取得と政策利用

民間企業向け

  • 駅構内・出口付近の広告展開
  • ルート別ターゲティングの可能性

チーム

TransitEaseは、交通データと人の行動理解を融合させるチームです。 UX設計・データ分析・都市交通の専門性を組み合わせ、 オープンデータの新たな価値を提案します。
