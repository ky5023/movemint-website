移動課題を内製コンポーネントで束ねる

「乗り場が分からない」「待ち時間が長い」「夜道が不安」「バリアフリー経路は？」といった現場課題を、MoveMint の三つのコンポーネントでリアルタイムに解決する。

MoveMint Map

駅構内トポロジー、ランドマーク座標、ジオフェンスを統合した動的マップであり、初訪問でも乗り場へ迷わず誘導できる設計とする。

  • 出口から乗り場までの3Dパスと位置補正ロジック
  • アクセシビリティタグと夜間安全スコアの重畳表示

Runtime Route Orchestrator

GTFS/GTFS-RT、遅延API、人流推定、天候・治安レイヤを掛け合わせ、乗り継ぎ成立確率と待機時間をリアルタイムで再計算するルートエンジンである。

  • 混雑度や天候に応じた余裕バッファの自動チューニング
  • ホーム間移動時間・改札混雑を実測データで補正

Local Support Chat AI

地域ごとの困りごとを理解する言語モデルであり、移動中の問い合わせをハンドリングしつつ、回答をマップとルートへ即反映する。

  • 自治体・交通事業者監修のローカルFAQコーパス
  • チャット問い合わせをトリガにルート再計算を発火

MoveMintが解決する移動課題

従来の経路アプリは「最短・最安」を答えるだけ。MoveMint は乗り場の迷子、不確かな安全情報、非現実的な乗り継ぎや待ち時間まで拾い上げ、移動前・途中・移動後の体験をまるごと最適化する。

目的ドリブンな経路提案

利用目的を起点に経路と体験モジュールを組み合わせる。テンプレートは目的別にバンドルし、初期設定を簡略化する。

  • 目的別テンプレートで入力負担を最小化
  • 乗り場・出口・集合場所を写真付きで推奨
  • 移動条件（時間帯・同行者・安全指標）をバリアントとして保持

移動時間を体験価値に変換

所要時間に合わせた音声・動画・記事を推薦し、滞在ステートに応じて体験カードを切り替える。完走結果は学習データとして回収する。

  • 移動時間にフィットする長さのコンテンツ
  • 乗車待ち余白向けマイクロアクティビティ
  • 視認性と安全ラインに沿ったUXガードレール

公共交通と連動する改善サイクル

移動後フィードバックで接続品質・混雑度・乗り換え体験を収集し、ダイヤ調整とサービス改善のためのインサイトとして出力する。

  • フィードバックは数タップで完了
  • 無理な乗り継ぎや待ち過ぎ、危険箇所をヒートマップ化
  • 公共交通事業者へKPIダッシュボードとして還元

利用の流れ

移動前・移動中・移動後の三フェーズを一つの体験に束ね、利用するたびに学習が蓄積するモビリティ体験を形成する。

1

移動前：プランニング

目的と条件を入力し、GTFS/GTFS-RT を組み合わせた最善ルートと滞在プランを生成する。

2

移動中：時間価値化

所要時間にフィットした音声・動画・記事を提示し、完了ログを学習データとして蓄積する。

3

移動後：フィードバック

到着状況や混雑感、乗り換え体験を数タップで記録し、公共交通への改善提案データとして吐き出す。

フィードバックデータは匿名化処理を通過し、サービス改善とインサイト開発に活用する。

MoveMintを支えるデータ基盤

オープンデータとリアルタイムデータ、さらに目的情報を掛け合わせ、現実に即したモビリティ体験を構築する。

Stage 01

GTFS（時刻表・経路）

ダイヤと経路データを解析し、目的別テンプレートに紐づく複数ルート候補を生成する。

Stage 02

GTFS-RT（リアルタイム）

遅延や運行状況をリアルタイムに反映し、所要時間と乗り換えリスクを即時補正して現実に合致させる。

Stage 03

行動ログ × フィードバック

移動中の選択と移動後の感想を照合し、推薦アルゴリズムと公共交通インサイトを継続更新する。

公共交通と地域へ還元されるインサイト

MoveMint Insight は移動データから意思決定に直結する指標を生成し、公共交通・自治体・事業者の共創を後押しする。

接続品質と混雑の可視化

Transfer Wait Index や Connection Reliability など、乗り換え体験を定量化する独自KPIを生成する。

  • ボトルネック区間の即時特定
  • 改善成果をダッシュボードで共有

目的別の需要把握

Purpose Mix Ratio で通勤・通学・観光など利用目的の変化を追跡し、施策効果を検証する。

  • イベント施策の効果測定
  • 目的別顧客体験スコアの算出

地域体験との連携

沿線の店舗・観光資源と接続し、移動時間を活用したキャンペーンや収益機会を創出する。

  • ポイント連動クーポンの配信
  • 回遊データを用いた施策PDCA

継続利用を促すリワード設計

利用者が移動体験に参加し続けるほど公共交通も賢くなる仕組みを整備する。ポイントは交通割引や沿線店舗のクーポンに交換可能であり、循環構造を維持する。

目的入力で 10pt

利用意図をデータ化し、時間帯や同行者と紐づけて個別最適化と都市施策に接続する。

コンテンツ完走で 20pt

移動を価値ある学びへ変換し、完走データでレコメンド精度を更新する。ポイントは提携コンテンツの割引券にも交換可能。

フィードバックで 30pt

到着後の所感を共有してポイントを還元し、公共交通の改善サイクルを駆動する。沿線キャンペーンや定期券アップグレードへの活用を想定。
